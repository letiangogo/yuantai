# Contribution Guidelines

Thank you for your interest in contributing to the Meta-State Cosmological Model!

## How to Contribute

### Submit an Issue
- Identify logical gaps or imprecise formulations
- Propose new observational test plans
- Discuss compatibility with existing theories

### Submit a Pull Request
- Fix typos or unclear expressions in the literature
- Supplement mathematical formalizations (equations, formal deductions)
- Translate into other languages

### Submit a Collaboration Proposal
- For major revisions or extensions to core concepts
- Send to **zhijundi@qq.com**
- The team will provide feedback within 15 working days

## Code of Conduct

- All discussions must be based on reason and logic; appeals to authority or ideology are rejected
- Constructive criticism takes priority over deconstructive questioning
- This model does not participate in any religious, political, or commercial ideological discussions

# 贡献指南

感谢你关注元态宇宙模型的共建工作！

## 如何贡献

### 提交 Issue
- 发现逻辑漏洞或表述不严谨之处
- 提出新的观测检验方案
- 讨论模型与现有理论的兼容性

### 提交 Pull Request
- 修正文献中的笔误或表述不清
- 补充数学化表达（方程、形式化推演）
- 翻译为其他语言

### 提交共建方案
- 对模型核心概念的重大修正或扩展
- 发送至 **zhijundi@qq.com**
- 团队将在15个工作日内反馈

## 行为准则

- 所有讨论须基于理性与逻辑，拒绝诉诸权威或意识形态
- 建设性批评优先于解构性质疑
- 本模型不参与任何宗教、政治或商业意识形态讨论
